# Smarty Address JS Plugin
All notable changes to this project will be documented in this file.

## 1.1.0 - 2026-05-22

CHANGES:

- Added support for the International Address Autocomplete v2 API, routed via the `country` and `countrySelector` config options.
- Renamed `AutocompleteSuggestion` fields to international terms.
- Fixed collapsed secondary suggestions for international addresses.
- Expanded test coverage for the international address flow.

## 1.0.1 - 2026-03-19

CHANGES:

- Patch release with minor fixes.

## 1.0.0 - 2026-03-16

CHANGES:

- Initial release of lightweight TypeScript library for adding Smarty address autocomplete to web forms. Available via npm and CDN (jsDelivr, unpkg).
